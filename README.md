# Brain Activity Visualizations

A collection of research-grade neuroscience visualizations modeling how external stimuli and environmental factors shape brain activity and neural responses.

Built with Python. Grounded in peer-reviewed neuroscience research.

---

## What This Project Models

The brain does not operate in isolation. Every perception, decision, and emotion is triggered by something outside it — a sound, a reward, a face, a threat. This repository visualizes those dynamics: how external input propagates through neural circuits, how the brain updates its predictions, and how activity patterns emerge from the interaction between organism and environment.

---

## Visualizations

### Dopamine Reward Prediction Error (RPE)
Models the firing patterns of dopamine neurons in response to expected and unexpected rewards, based on Schultz, Dayan & Montague (1997). Includes raster plots, peri-stimulus time histograms (PSTH), and temporal difference signal decomposition.
